Corsican Language

Introduction to Corsican

Corsican is a Romance language spoken on the island of Corsica. It shares many similarities with Italian, particularly with the dialects of Tuscany. It is considered a vulnerable language as it faces competition from the dominant French language in the region.

Language Details

ISO Code cos
Dialects Corsu, Cismuntincu, Suttanaisu
Number of Speakers 150,000 - 200,000
Language Status Vulnerable
Language Family Romance, Italo-Dalmatian
