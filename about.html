About This Project

Introduction

The world faces a critical crisis as many languages are on the brink of extinction. The endangerment of languages is a significant issue affecting humanity, presenting moral, practical, and scientific challenges. This website focuses on documenting endangered languages in Europe, complementing existing resources such as Endangered Languages Project and Ethnologue . Our goal is to provide detailed information to raise awareness, preserve linguistic diversity, and enhance our understanding of human language.

Language Information

The data on this website is sourced from Endangered Languages Project, specifically curated and refined to focus on Europe. Each language page on the website provides comprehensive details, including:

  • ISO Code: The ISO 639-3 code uniquely identifies the language.
  • Dialects: Lists the various names and dialects used to refer to the language or its variants.
  • Number of Speakers: Indicates the estimated number of speakers using the language within the country.
  • Language Family: Classifies the language into a broader language family, such as Indo-European or Uralic.
  • Language Status: Reflects the level of endangerment as defined by UNESCO's Language Endangerment Scale. Each language is categorized from "At Risk" to "Critically Endangered," based on criteria like intergenerational transmission and the absolute number of speakers. These assessments are subject to change as new information becomes available.

Each language page features an interactive map highlighting the specific regions where the language is spoken. This visual representation helps users understand the geographical context of the language's use. Below the map, the page presents a list of common words and phrases in the language, offering a glimpse into its vocabulary and phonetic characteristics. These phrases are designed to give users a sense of how the language sounds and its unique linguistic features.
